# Nexo.js

Crie bots Discord modernos usando TypeScript ou JavaScript,
com menos boilerplate e uma arquitetura pronta para escalar.

## 🧭 O que é o Nexo.js?

O Nexo.js é um framework construído sobre o discord.js
que fornece estrutura, carregamento automático e uma CLI integrada
para acelerar o desenvolvimento de bots Discord.

## ✨ Por que usar o Nexo.js?

- ⚡ Setup rápido — comece em minutos
- 🧱 Estrutura pronta para projetos reais
- 🧰 CLI integrada para desenvolvimento
- 🔌 Compatível com Node.js e Bun
- 🧠 Arquitetura escalável

## 🎯 Para quem é o Nexo.js?

- Desenvolvedores iniciando bots Discord
- Projetos que precisam crescer sem virar bagunça
- Quem quer menos configuração e mais produtividade

## 📦 O que você vai encontrar aqui

- Como instalar e rodar seu bot
- Como criar comandos e eventos
- Como organizar seu projeto
- Exemplos práticos
