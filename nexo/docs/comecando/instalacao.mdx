# Instalação

## 📥 Instalar

```bash
npm install nexocord discord.js
```

## ⚙️ CLI do Nexo

O Nexo inclui uma CLI para criar e executar projetos com a estrutura padrão do framework. Ela prepara o projeto inicial para você começar sem montar a base manualmente.

## 🚀 Criar projeto

```bash
npx nexo create meu-bot
```

:::info Importante
O comando `create` gera a base do projeto com os arquivos e pastas iniciais já prontos para o próximo passo.
:::
